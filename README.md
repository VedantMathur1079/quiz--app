# ⚡ QuizApp — Version Control System Project

## 📌 Overview

This project is a browser-based quiz application developed as part of a Version Control System (VCS) project. It demonstrates the use of Git and GitHub for tracking changes, managing versions, and collaborating effectively.

## 🚀 Features

* Interactive quiz interface
* Multiple-choice questions
* Timer functionality
* Score tracking
* Leaderboard system
* Clean and responsive UI

## 🛠️ Tech Stack

* HTML (Single-file application)
* JavaScript (Embedded)
* CSS (Embedded)

## 📁 Project Structure

```
quiz-app/
│── QuizApp.html
│── README.md
│── CONTRIBUTING.md
│── .gitignore
```

## 🔄 Version Control Usage

* Initial project created with multiple files
* Refactored into a single optimized HTML file
* Git used for tracking changes and commits
* Branching used for feature development
* GitHub used for repository management
